## üéØ Game Mechanics

### Controls
- **Desktop**: Arrow keys for movement, Space for handbrake, R to reset
- **Mobile**: Touch controls with intuitive directional buttons

### Objectives
1. Explore the 3D landscape to find hidden energy nodes
2. Collect 30 nodes to complete the exploration
3. Avoid obstacles that can cost collected nodes
4. Unlock portfolio sections progressively
5. Complete the experience to view final completion time

### Features
- **Real-time HUD** - Speed, node count, progress, and radar information
- **Visual Feedback** - Notifications for node collection and section unlocks
- **Interactive Elements** - Holographic portfolio panels with smooth animations
- **Time Tracking** - Completion time display for competitive exploration
